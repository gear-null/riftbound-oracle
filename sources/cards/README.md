# sources/cards

Place card data files here for processing.

## Expected formats

- **JSON** — API dumps from Riftcodex or similar (`.json`)
- **HTML** — Saved web pages from card databases (`.html`)

## Notes

- The `fetch` command pulls card data directly from the Riftcodex API, so you typically won't need to place files here manually.
- Files in this directory (except this README) are gitignored.
